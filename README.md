# 📊 Análise de Dados Educacionais — Fundamentos de IA

Repositório desenvolvido para o curso **Fundamentos de IA e Análise de Dados**, contendo notebooks, scripts e datasets para treinamento prático em análise exploratória, visualização e modelagem estatística aplicada à educação.

---

## 📦 Sobre os Dados

As bases utilizadas foram extraídas do site **[1]** e, em alguns casos, adaptadas para fins didáticos.

> ⚠️ **Aviso importante:** Os datasets contêm uma mistura de dados reais e fictícios. Sua reprodução e modificação são **totalmente livres**, inclusive para fins acadêmicos e comerciais.

---

## 🛠️ Tecnologias Utilizadas

| Biblioteca | Finalidade |
|---|---|
| `pandas` | Manipulação e análise de dados tabulares |
| `numpy` | Computação numérica e operações vetoriais |
| `seaborn` | Visualização estatística de dados |
| `matplotlib` | Gráficos complementares |
| Jupyter Notebook | Prototipagem e documentação das análises |

Todos os scripts seguem as boas práticas de **clean code** e estão formatados conforme o **[2]**.

---

## ⚡ Como Começar

### 1. Instalar as dependências

```bash
pip install pandas numpy seaborn matplotlib jupyter
```

### 2. Estrutura do repositório

```
.
├── notebooks/          # Análises exploratórias em formato .ipynb
├── scripts/            # Scripts Python auxiliares
├── data/               # Arquivos CSV dos datasets utilizados
└── README.md           # Você está aqui 👋
```

### 3. Executar uma análise

Clone o repositório e inicie o Jupyter:

```bash
git clone [3]
cd analise-dados-educacionais
jupyter notebook
```

Em seguida, abra os notebooks na pasta `notebooks/` e execute as células sequencialmente.

---

## 📊 Exemplo de Análise

As análises incluem:

- **Correlação entre hábitos de estudo e desempenho acadêmico**
- **Regressão linear múltipla** para identificar fatores preditivos
- **Visualizações estatísticas** com Seaborn (scatter plots, boxplots, bar charts)
- Segmentação de alunos por faixas etárias, socioeconômicas e comportamentais
